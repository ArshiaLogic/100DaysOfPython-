# 🚀 100 Days of Code: The Complete Python Pro Bootcamp
### My Journey through Dr. Angela Yu's Python Masterclass

Welcome to my Python learning repository! 🐍 🎸 ♟️

This repository serves as a personal log and portfolio for the **100 Days of Code Challenge**. My goal is to master Python from the ground up, moving from basic scripts to complex applications involving Web Development, Data Science, and AI.

## 📌 Project Overview
The course, taught by [Dr. Angela Yu](https://www.udemy.com/course/100-days-of-code/), is a comprehensive bootcamp covering:
- **Core Python:** Variables, Logic, Loops, and Functions.
- **OOP:** Object-Oriented Programming principles.
- **GUI & Games:** Turtle, Tkinter, and game logic.
- **Web Development:** Flask, Jinja, and REST APIs.
- **Data Science:** Pandas, NumPy, Selenium, and Scrapy.
- **Automation:** Scripting for productivity.
